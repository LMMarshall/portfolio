Global model for cost forecasting

Forecasting costs across thousands of separate time series.

Objective. Build a single model capable of accurately forecasting costs using multivariate data across 2000+ separate time series.

Approach:

  • PySpark in Azure Databricks for data extraction/transformation pipeline
  • Handle gaps in time series
  • Handle multiple currencies and countries
  • Used DARTS Temporal Fusuion Transformer (TFT) model, which is able to handle multiple time series and allows for probabilistic forecasting

Challenges:

  • Extensive data cleaning and reconciliation of conflicting data points (e.g., transactions that replaced earlier transactions; transactions that supplemented earlier transactions, etc)

Outcomes:

  • The process of reviewing the data before modeling exposed a number of faulty assumptions in the original mode
  • The new TFT model reduces loss by 23% over previous exponential smoothing model (even after the faulty assumptions in the earlier model were corrected for)
