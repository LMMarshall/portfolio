Global model for cost forecasting

Forecasting costs across thousands of separate time series.

Objective. Build a model capable of accurately forecasting costs using multivariate data across 2000+ time series. Specifically, this model was intended to replace an existing exponential smoothing model that performed poorly.

Approach:

  • Interviewed stakeholders to understand logic for reconciling duplicate and supplemental pay events
  • Built data extraction/transformation pipeline with PySpark in Azure Databricks
  • Applied conditional filtering on data during ETL process for data privacy
  • Handled currency conversions across multiple countries
  • Used DARTS Temporal Fusion Transformer (TFT) to build a forecasting model
  • Compared TFT results to existing exponential smoothing results and a naive baseline model and tested for significance

Challenges:

  • Extensive data cleaning and reconciliation of conflicting data points (e.g., transactions that replaced earlier transactions; transactions that supplemented earlier transactions, etc) as well as different pay freqencies across time series

Outcomes:

  • In my review and exploration of the data before modeling, I uncovered a number of faulty assumptions in the original exponential smoothing model. This finding alone was a significant win, since it exposed errors that were then correctable
  • The new TFT model reduced loss by 23% over the exponential smoothing model (even after the faulty assumptions in the earlier model had been corrected for)
