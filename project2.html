Seattle Police: Time-series forecasting

A big data project using PySpark with Google Cloud, OpenRefine for data cleaning, extensive feature engineering, ARIMA forecasting

Objective. The Seattle Police Department faced substantial challenges in managing its policing resources in 2020 when it was one of the first (and hardest-hit) U.S. cities to face COVID-19 and shortly afterwards was hit by mass Black Lives Matter protests, which led to the forced shutdown of East Precinct station and calls to defund the police.

The goal of this project was to use police call data from the City of Seattle's Open Data project to:

  1. Build a machine-learning model that would predict 2020 police call acitivity based on 10 years of historical data. This would provide a baseline by which to measure the impact of COVID and the protest.
  2. Compare that expected outcome with the actual police call activity for 2020 to gauge the impact of these events on police resource utilization.
  3. Construct a tool that would allow police and public users to interact with this data and easily view the impacts at various levels of granularity.

Approach details:

  • 4.3 million rows of police call data with 11 features
  • Used OpenRefine to fix tagging inconsistencies in the data and aggregate similar categories
  • Used Google Cloud + PySpark for feature engineering, cleaning, and aggretating data
  • Used Python Jupyter notebook for final cleaning
  • Built 45 seasonal ARIMA models to support forecasting (9 features X 5 precincts) and used ADF test for stationarity
  • Wrote accessible feature descriptions for users to provide critical context on how each feature had been built from the underlying data.
Above: Simple feature descriptions for users preserve critical context.

Results:<\b>

The resulting tool surfaced a number of interesting insights that could help inform police resource planning during times of social upheaval. For example:

  • The proportion of calls resulting in no police action surged from Jan to Aug by 35%, potentially costing time and resources that could have been allocated elsewhere
  • Average police response time increased by 87% to from Jan to Aug, 2020
  • Arrest numbers across the city dropped by 21% from Jan to April, 2020 (the initial months of COVID-19 before the BLM protests began). During the same period, total police action declined by almost 10%.
  • The ratio of public calls for help to police-initiated calls increased by 257% between Jan and Aug 2020 just as calls poured in to defund the police. The East precinct saw a much steeper increase of 482%.
